How it works (high level)

  1. User chats via WhatsApp
  2. NetSeA classifies intent (rules + AI fallback)
  3. Missing inputs are requested (only if needed)
  4. NetSeA calls the correct backend module (Node/Python/DB)
  5. Bot replies with insights + attachments

Example triggers: hi netsea, kpi trend, geo coverage, complaint, site info

Modules (public overview)

  • Complaint Copilot: extracts key fields (e.g., timeframe, location, description) and runs network checks.
  • KPI Trendline: hourly/daily KPI charts + caption summary for quick interpretation.
  • Geo Coverage: signal/quality snapshot and neighbor context for investigation.
  • Network Summary: aggregates multiple checks and generates a concise health summary.
  • Site Info: key reference details + nearby site context.
